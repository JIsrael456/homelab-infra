# Fase 01 - Infraestrutura de Rede

## Objetivo

Planejar e configurar a infraestrutura de rede que será utilizada por todos os servidores e estações da empresa.

## Tecnologias

- VirtualBox/VMware
- IPv4
- DNS
- Gateway
- DHCP
- NAT/Bridge

## O que será feito

- [ ] Definir arquitetura da rede
- [ ] Criar rede virtual
- [ ] Definir faixa de IPs
- [ ] Configurar Gateway
- [ ] Configurar DNS
- [ ] Testar comunicação entre máquinas
- [ ] Documentar topologia

## Resultado esperado

Toda a infraestrutura será capaz de se comunicar através da rede interna da empresa.
